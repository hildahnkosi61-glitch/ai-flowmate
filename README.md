# AI Workplace Productivity Assistant

A modern, responsive web application designed to help professionals automate everyday workplace tasks using Artificial Intelligence. The application provides an intuitive dashboard with AI-powered productivity tools that improve efficiency, organization, and communication.

---

## Project Overview

The **AI Workplace Productivity Assistant** is a SaaS-inspired productivity platform that combines multiple AI-powered workplace tools into a single, user-friendly interface. It features a clean dashboard layout, responsive design, and structured prompt engineering to generate professional and reliable outputs.

---

## Featutures 
📧 Smart Email Generator

Generate professional emails tailored to different situations.

**Capabilities:**

* Tone selection (Professional, Friendly, Formal, Casual)
* Audience-based customization
* AI-generated email subject and body
* Clear and professional formatting

---

### 📝 Meeting Notes Summarizer

Convert lengthy meeting notes into concise summaries.

**Outputs include:**

* Key discussion points
* Action items
* Deadlines
* Assigned responsibilities (where applicable)
